# 💃 Insane Dance Center App

Insane Dance Center App este o aplicație mobilă creată pentru administrarea unui studio de dans.

Proiectul este dezvoltat inițial pentru Insane Dance Center și este construit astfel încât să poată fi adaptat ulterior pentru orice școală de dans.

---

## Obiective

- Calendar comun
- Managementul grupelor
- Managementul elevilor
- Prezențe
- Competiții
- Rezultate
- To-Do
- Anunțuri
- Rapoarte

---
